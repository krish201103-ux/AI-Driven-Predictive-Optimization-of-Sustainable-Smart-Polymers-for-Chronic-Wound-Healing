# AI-Driven Predictive Optimization of Sustainable Smart Polymers for Chronic Wound Healing
## Overview
This project uses Artificial Intelligence and Machine Learning approaches to analyze and optimize sustainable smart polymers for chronic wound healing applications.

The project integrates:
- Random Forest Regression
- Deep Learning (MLP)
- XGBoost
- SHAP Explainability
- Feature Importance Analysis
- Polymer Combination Optimization

---

## Objectives
- Predict the healing efficiency of polymers
- Rank polymers based on wound healing capability
- Compare AI models for prediction accuracy
- Identify best polymer combinations for lab synthesis
- Generate explainable AI insights using SHAP

---

## Dataset Features
- Molecular Weight (MW)
- Glass Transition Temperature (Tg)
- Contact Angle
- Degradation Rate
- Tensile Strength
- Drug Release Percentage
- Bioactivity Score
- Antimicrobial Activity

---

## Models Used
| Model | Purpose |
|------|------|
| Random Forest | Ensemble Learning |
| Deep Learning (MLP) | Nonlinear Prediction |
| XGBoost | Gradient Boosting Optimization |

---

## Explainable AI
SHAP analysis is used to determine:
- Most influential polymer properties
- Global feature importance
- Healing efficiency drivers

---

## Key Findings
- PCL-based nanocomposites showed the highest healing efficiency
- Degradation rate strongly influences wound healing
- Drug release and antimicrobial activity are major predictors
- XGBoost demonstrated strong predictive consistency
